# EasyConverter - приложение для конвертации валют с использованием Flask. 
## Веб-приложение предоставляет возможность получения актуальных курсов валют и конвертации между ними.
### Функционал
### 1.	Получение актуальных курсов валют: 
### –  Реализована функция, запрашивает актуальные курсы валют с внешнего API https://exchangeratesapi.io/ и сохраняет их в базе данных. ;### – Реализована функция, которая возвращает дату и время последнего обновления курсов в базе данных.
### 2.	Конвертация между валютами:
### 3.	– Реализована функция, которая принимает две валюты и сумму для конвертации. Функция возвращает результат конвертации на основе актуальных курсов валют из базы данных. Работает через Ajax-запросы.
### 4.	Использованный стек: – Flask
## Реализовано три эндпоинта:
### * Обновление курсов валют в БД на актуальные курсы.
### * Получение даты и времени последнего обновления курсов в БД.
### *Конвертация между валютами. – Валидация форм и отправка данных через Ajax-запросы.
## Интерфейс: базовый HTML и CSS + Bootstrap.
